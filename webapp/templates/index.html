Log Failure Prediction using LSTM

Paste raw logs, map them to EventIds, and predict failures in one flow.

The system parses each log line, matches it to an HDFS template, builds the last {{ sequence_length }} EventIds, and sends the sequence to the trained LSTM model.

Model focus
Sequence-based anomaly detection
  • Template ID mapping from raw logs
  • Unknown event detection
  • Alert severity classification
  • Root cause event hint

Input Logs

See example

What Happens

1

Read raw log lines from the browser form.

2

Match each line to the closest HDFS template.

3

Convert matches into EventId sequence.

4

Predict failure/anomaly using the trained LSTM.
